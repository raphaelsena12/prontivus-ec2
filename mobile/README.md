# Prontivus Mobile App

Aplicativo mobile desenvolvido em Flutter com SQLite para o sistema Prontivus.

## Tecnologias

- **Flutter**: Framework para desenvolvimento mobile multiplataforma
- **SQLite**: Banco de dados local para armazenamento offline

## Estrutura do Projeto

```
mobile/
├── lib/
│   ├── main.dart
│   ├── models/
│   ├── services/
│   ├── screens/
│   └── utils/
├── pubspec.yaml
└── README.md
```
